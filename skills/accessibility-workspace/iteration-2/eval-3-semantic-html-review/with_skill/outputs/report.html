목록 구조 (ul/ol/li) ⚠️ 정적분석 Navbar.tsx 33~46행: 반복 렌더링된 navLinks를 div 컨테이너 안 개별 <Link>로 나열. 의미상 내비게이션 링크 목록이므로 <ul>/<li> 구조가 더 적합. nav 링크를 리스트로 감싸기:

<fieldset><legend>필터</legend>...</fieldset>
