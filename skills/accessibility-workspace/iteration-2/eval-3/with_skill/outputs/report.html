시맨틱 HTML 검토

검토 항목결과판정방식발견된 문제권고 수정
랜드마크 요소 (header/main/footer/nav) ❌ 정적분석 <header>, <main>, <footer>, <nav> 랜드마크 요소가 전혀 사용되지 않음. <body> 내부에 제목, 링크, 이미지, 본문 텍스트가 직접 나열됨. 페이지 구조를 <header>(로고/네비게이션), <main>(본문 콘텐츠), <footer>(하단 정보)로 구분. 링크 영역은 <nav>로 감싸기.
섹션 구조 (article/section/aside) ⚠️ 정적분석 콘텐츠 블록이 <section> 또는 <article> 없이 평면적으로 나열됨. 프로모션 영역 등 독립적 콘텐츠 블록 구분이 부족함. 독립적인 콘텐츠 영역(히어로 배너, 프로모션 등)을 <section>으로 감싸고 적절한 제목(<h2>) 부여 권고.
대화형 요소 (button/a vs div/span) ✅ 정적분석 링크는 모두 네이티브 <a> 태그로 적절히 사용됨. 클릭 가능한 <div>/<span> 없음.
목록 구조 (ul/ol/li) ➖ 정적분석 목록으로 표현해야 할 반복 항목 없음
표현용 태그 (b/i, 연속 br) ➖ 정적분석 <b>, <i> 단독 사용 및 연속 <br><br> 패턴 없음
폼 그룹 (fieldset/legend) ➖ 정적분석 폼 요소 없음
기타 시맨틱 요소 (time/blockquote 등) ➖ 정적분석 인용문, 날짜/시간 표시 없음

불합격 (❌)

항목 1. 적절한 대체 텍스트 — index.html:30

문제: <img src="/public/logo.png" width="120" height="40" />에 alt 속성이 누락됨. 파일명으로 보아 로고 이미지로 추정되며, 스크린 리더가 이미지 내용을 전달할 수 없음.

수정: <img src="/public/logo.png" alt="사이트 로고" width="120" height="40" />

항목 17. 반복 영역 건너뛰기 — index.html (전체)

문제: 건너뛰기 내비게이션 링크가 없음. <main> 랜드마크도 없어 키보드 사용자 및 스크린 리더 사용자가 반복 영역을 건너뛸 수 없음.

수정:

<body>
  <a href="#main-content" class="skip-nav">본문 바로가기</a>
  <!-- 헤더/네비게이션 영역 -->
  <main id="main-content">
    <!-- 본문 콘텐츠 -->
  </main>
</body>

시맨틱 HTML: 랜드마크 요소 — index.html (전체)

문제: <header>, <main>, <footer>, <nav> 등 HTML5 랜드마크 요소가 전혀 사용되지 않음. 보조 기술 사용자가 페이지 구조를 파악하기 어려움.

수정: 페이지를 <header>, <nav>, <main>, <footer>로 구조화. 예시:

<body>
  <header>
    <h1>Welcome to the Sample Page</h1>
    <nav><a href="https://example.com/about">About Us</a></nav>
  </header>
  <main id="main-content">
    <!-- 이미지, 본문 콘텐츠 -->
  </main>
</body>

권고 (⚠️)

항목 18. 제목 제공

내용: <title>Sample Page</title>은 페이지 내용을 구체적으로 설명하지 않음. "Sample Page"는 임시 제목으로, 실제 서비스에서는 페이지 내용을 반영한 구체적인 제목(예: <title>Welcome - 서비스명</title>)으로 변경 권고. 또한 하위 제목 계층(<h2> 등)을 추가하여 콘텐츠 블록을 구분하면 접근성이 향상됨.

시맨틱 HTML: 섹션 구조

내용: 콘텐츠가 평면적으로 나열되어 있음. 히어로 배너 영역, 프로모션 영역 등을 <section>으로 구분하고 각 섹션에 제목을 부여하면 문서 구조가 명확해짐.
