불합격 (✗)

항목 29. 레이블 제공 — SearchFilter.tsx:21

문제: 검색 입력 필드(<input type="search">)에 <label>, aria-label, title 속성이 없음. placeholder="할 일 검색..."만 단독 사용. KWCAG 기준 placeholder는 레이블로 인정되지 않음.

수정: <input type="search" aria-label="할 일 검색" placeholder="할 일 검색..." />

항목 33. 웹 애플리케이션 접근성

문제 1: Navbar.tsx:59 — 햄버거 메뉴 버튼에 aria-expanded 속성 누락. 메뉴 열림/닫힘 상태를 보조 기술이 인식 불가. Playwright 검증으로 확인: 클릭 전후 ARIA 상태 변경 없음.

수정: <button aria-expanded={menuOpen} aria-label={menuOpen ? "메뉴 닫기" : "메뉴 열기"}>

문제 2: SearchFilter.tsx:32 — 필터 토글 버튼에 aria-pressed 속성 누락. 현재 선택된 필터를 CSS 클래스(bg-blue-600)로만 구분하며, 스크린 리더 사용자는 선택 상태를 알 수 없음.

수정: <button aria-pressed={filter === f.value}>{f.label}</button>
