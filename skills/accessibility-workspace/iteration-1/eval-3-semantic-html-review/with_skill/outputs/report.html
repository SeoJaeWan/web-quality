시맨틱 HTML 검토

검토 항목결과판정방식발견된 문제권고 수정
랜드마크 요소 (header/main/footer/nav) ❌ 정적분석 페이지에 <header>, <main>, <footer>, <nav> 랜드마크 요소가 전혀 사용되지 않음. <body> 안에 h1, 이미지, 링크, 스크립트가 직접 나열되어 있어 스크린 리더가 페이지 구조를 파악할 수 없음. <header>로 상단 영역, <main>으로 본문 영역, <footer>로 하단 영역을 감싸기. 내비게이션 링크가 있다면 <nav> 사용.
섹션 구조 (article/section/aside) ⚠️ 정적분석 독립적 콘텐츠 블록(히어로 영역, 프로모션 영역)이 <section>이나 <article>로 구분되지 않음 의미적으로 구분되는 콘텐츠 블록은 <section>으로 감싸고 헤딩 제공
대화형 요소 (button/a vs div/span) ✅ 정적분석 클릭 가능한 요소가 적절히 <a> 태그로 구현됨. 클릭 이벤트가 있는 <div>/<span> 없음.
목록 구조 (ul/ol/li) ➖ 정적분석 반복 항목이 없어 목록 구조 불필요
표현용 태그 (b/i vs strong/em) ➖ 정적분석 <b>, <i> 태그 사용 없음
폼 그룹 (fieldset/legend) ➖ 정적분석 폼 요소 없음
기타 시맨틱 요소 (time/blockquote 등) ➖ 정적분석 날짜/시간 표시, 인용문 없음

불합격 (❌)

항목 1. 적절한 대체 텍스트 — index.html:30

문제: <img src="/public/logo.png" width="120" height="40" />에 alt 속성이 완전히 누락됨. 로고 이미지는 의미 있는 콘텐츠이므로 대체 텍스트가 반드시 필요.

수정: <img src="/public/logo.png" alt="사이트 로고" width="120" height="40" />

항목 17. 반복 영역 건너뛰기 — index.html:20-45

문제: 페이지에 건너뛰기 링크가 없으며, <main> 랜드마크도 없음. 키보드/스크린 리더 사용자가 반복 영역을 건너뛸 수 없음.

수정:

<body> 최상단에 추가:

<a href="#main-content" class="skip-nav">본문 바로가기</a>

본문 영역을 <main id="main-content">으로 감싸기

시맨틱 HTML: 랜드마크 요소 — index.html:20-45

문제: <header>, <main>, <footer>, <nav> 랜드마크 요소가 전혀 사용되지 않음. 페이지 구조가 평면적이어서 보조 기술이 영역을 구분할 수 없음.

수정: 의미에 따라 <header>, <main>, <footer>로 영역 구분

권고 (⚠️)

항목 9. 콘텐츠 간의 구분

내용: 인접한 콘텐츠(이미지, 링크, 텍스트)의 시각적 구분이 인라인 CSS에 명시되지 않음. 외부 CSS(styles.css)에서 처리될 수 있으나, 외부 CSS 로드 실패 시 구분 불가. 콘텐츠 블록 간 margin, padding, border 등으로 명확한 시각적 구분 권고.

시맨틱 HTML: 섹션 구조

내용: 히어로 배너, 프로모션 배너, 텍스트 콘텐츠 등 의미적으로 구분되는 영역이 <section>이나 <article>로 감싸지지 않음. 각 콘텐츠 블록을 <section>으로 감싸고 적절한 헤딩을 제공하면 페이지 구조 이해도가 향상됨.
