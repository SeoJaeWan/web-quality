## 원칙 1. 인식의 용이성 (Perceivable)

### 1. 적절한 대체 텍스트 제공 — ❌ 위반 (정적분석)

**발견된 문제:**
- `index.html:30` — `<img src="/public/logo.png" width="120" height="40" />` 에 `alt` 속성이 누락되어 있음. 로고 이미지는 의미 있는 콘텐츠이므로 대체 텍스트가 필수임.

**수정 가이드:**
```html
<img src="/public/logo.png" alt="사이트 로고" width="120" height="40" />
```

## 원칙 2. 운용의 용이성 (Operable)

### 10. 키보드 사용 보장 — ✅ 준수 (정적분석)

페이지 내 인터랙티브 요소(`<a>` 링크)는 네이티브 HTML 요소를 사용하고 있어 키보드 접근이 보장됨. `onClick`이 있는 비대화형 요소(`<div>`, `<span>`)는 발견되지 않음.

단, `index.html:39-43`의 스크립트가 `getElementById("label")`로 요소를 조작하나, 해당 요소(`id="label"`)가 HTML에 존재하지 않으므로 실질적 키보드 문제는 없음.

---

### 11. 초점 이동과 표시 — ✅ 준수 (정적분석)

CSS에서 `outline: none` 또는 `outline: 0` 패턴이 발견되지 않음. `tabIndex={-1}` 남용이나 `onfocus="this.blur()"` 패턴 없음.

---

### 12. 조작 가능 — ✅ 준수 (정적분석)

링크 요소가 기본 크기로 제공됨. 명시적으로 매우 작은 크기를 설정한 컨트롤이 없음.

---

### 13. 문자 단축키 — ➖ 해당없음

`addEventListener('keydown')` 등 키보드 단축키 관련 코드가 없음.

---

### 14. 응답시간 조절 — ➖ 해당없음

`setTimeout`, `setInterval` 등 시간 제한 관련 코드가 없음.

---

### 15. 정지 기능 제공 — ➖ 해당없음

자동 변경 콘텐츠(슬라이드, 배너 등)가 없음.

---

### 16. 깜박임과 번쩍임 사용 제한 — ✅ 준수 (정적분석)

깜박임이나 번쩍임을 유발하는 CSS animation 또는 콘텐츠가 없음.

---

### 17. 반복 영역 건너뛰기 — ❌ 위반 (정적분석)

**발견된 문제:**
- 건너뛰기 링크(`<a href="#main">`, `<a href="#content">` 등)가 페이지 최상단에 제공되지 않음.
- `<main>` 랜드마크 요소도 없어 건너뛰기 대상 자체가 불명확함.

**수정 가이드:**
```html
<body>
  <a href="#main-content" class="skip-nav">본문 바로가기</a>
  <!-- 반복 영역 (헤더, 내비게이션 등) -->
  <main id="main-content">
    <!-- 본문 콘텐츠 -->
  </main>
</body>
```

### 18. 제목 제공 — ⚠️ 권고 (정적분석)

- 헤딩 구조: `<h1>Welcome to the Sample Page</h1>` 하나만 존재. 콘텐츠 블록별 하위 헤딩(`<h2>` 등)이 없음.

**수정 가이드:**
- `<title>` 을 "페이지명 - 서비스명" 형식으로 구체적으로 작성
- 콘텐츠 블록에 `<h2>`, `<h3>` 등 하위 헤딩 추가 권고

**발견된 사항:**

## 시맨틱 HTML 주요 위반 상세

### 랜드마크 누락 — `index.html` 전체

**문제:**
현재 페이지는 `<body>` 하위에 `<h1>`, `<a>`, `<img>`, `<p>`, `<script>` 등이 평면적으로 나열되어 있으며, 페이지 구조를 나타내는 시맨틱 랜드마크 요소가 전혀 없음.

**권장 구조:**
```html
<body>
  <a href="#main-content" class="skip-nav">본문 바로가기</a>
  <header>
    <img src="/public/logo.png" alt="사이트 로고" width="120" height="40" />
    <nav>
      <a href="https://example.com/about">About Us</a>
    </nav>
  </header>
  <main id="main-content">
    <h1>Welcome to the Sample Page</h1>
    <img src="/public/hero.png" alt="Hero banner" width="800" height="400" />
    <img src="/public/promo.png" alt="Promotional banner" />
    <p>Check out our <a href="https://example.com/deals">latest deals</a>.</p>
  </main>
  <footer>
    <!-- 푸터 콘텐츠 -->
  </footer>
</body>
```
