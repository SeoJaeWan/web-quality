항목 1. 적절한 대체 텍스트 — index.html:30

문제: <img src="/public/logo.png" width="120" height="40" />에 alt 속성이 누락되어 스크린 리더가 이미지 내용을 전달할 수 없음

수정: <img src="/public/logo.png" alt="사이트 로고" width="120" height="40" />

항목 17. 반복 영역 건너뛰기 — index.html

문제: 페이지에 건너뛰기 링크(skip navigation)가 없어 키보드 사용자가 반복 영역을 건너뛸 수 없음

수정: <body> 최상단에 <a href="#main-content" class="skip-nav">본문 바로가기</a> 추가, 본문에 id="main-content" 부여

항목 23. 레이블과 네임 — dashboard.tsx:29-36

문제: SVG 아이콘만 포함된 버튼 2개에 aria-label이 없어 스크린 리더가 버튼 용도를 전달할 수 없음

수정: <button aria-label="알림"><svg>...</svg></button>

항목 32. 마크업 오류 방지 — Card.tsx:27, dashboard.tsx:46

문제: (1) Card 컴포넌트에 id="card-item" 하드코딩으로 리스트 렌더링 시 중복 ID 발생. (2) items.map()에서 key prop 미제공

수정: (1) 고유 ID 사용 id={`card-item-${id}`} 또는 id 제거. (2) <div key={item}>로 key 추가

시맨틱 HTML: 랜드마크 요소 — index.html

문제: <body> 직하에 랜드마크 요소 없이 콘텐츠가 배치되어 있어 스크린 리더의 영역 탐색이 어려움

수정: <header>, <main>, <footer>로 페이지 구조 구분

시맨틱 HTML: 대화형 요소 — Card.tsx:24-29

문제: 클릭 가능한 <div onClick>가 <button> 대신 사용됨. 키보드 접근 불가, 스크린 리더에서 역할 인식 불가

수정: <button> 요소로 변경하거나 role="button" tabIndex={0} onKeyDown={handler} 추가
