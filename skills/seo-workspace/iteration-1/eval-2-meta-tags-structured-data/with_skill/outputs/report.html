Fix Guide

SEO-02: title tag too short and generic

File: index.html, line 6

Current: <title>Sample Page</title> (11 chars)

Fix: Replace with a descriptive, keyword-rich title of 50-60 characters.

<title>Welcome to Example Store | Premium Products & Free Shipping</title>

SEO-05: Missing meta description

File: index.html, <head> section

Fix: Add a compelling meta description of 150-160 characters.

<meta name="description" content="Discover premium products with free shipping and 30-day returns. Shop our latest deals and save up to 20%. Trusted by thousands of customers.">

SEO-06: Missing canonical URL

File: index.html, <head> section

Fix: Add a self-referencing canonical link.

<link rel="canonical" href="https://example.com/">

SEO-07: Missing structured data (JSON-LD)

File: index.html, <head> section

Fix: Add JSON-LD structured data. For a homepage, use Organization or WebSite schema.

<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Organization",
  "name": "Example Store",
  "url": "https://example.com",
  "logo": "https://example.com/public/logo.png"
}
</script>

WP-01: Render-blocking inline script

File: index.html, lines 37-44

Fix: Move the script to an external file and add defer, or wrap the logic in a DOMContentLoaded listener and place the script tag with defer.

<script defer src="/js/main.js"></script>

WP-02: LCP image missing fetchpriority and preload

File: index.html, line 27 and <head>

Fix: Add fetchpriority="high" to the hero image and a preload link in the head.

<!-- In <head> -->
<link rel="preload" href="/public/hero.png" as="image" fetchpriority="high">

<!-- On the img tag -->
<img src="/public/hero.png" alt="Hero banner" width="800" height="400" fetchpriority="high">

WP-03: Render-blocking external stylesheet

File: index.html, line 10

Fix: Inline critical above-fold CSS and defer the external stylesheet.

<style>/* Inline critical above-fold CSS here */</style>
<link rel="preload" href="https://cdn.example.com/styles.css" as="style" onload="this.onload=null;this.rel='stylesheet'">
<noscript><link rel="stylesheet" href="https://cdn.example.com/styles.css"></noscript>

WP-04: Image missing width/height (CLS risk)

File: index.html, line 33

Current: <img src="/public/promo.png" alt="Promotional banner">

Fix: Add explicit width and height attributes.

<img src="/public/promo.png" alt="Promotional banner" width="600" height="300">

WP-05: Below-fold images missing loading="lazy"

File: index.html, lines 30, 33

Fix: Add loading="lazy" to non-LCP images.

<img src="/public/logo.png" width="120" height="40" loading="lazy">
<img src="/public/promo.png" alt="Promotional banner" width="600" height="300" loading="lazy">

WP-06: No modern image formats

File: index.html, all <img> tags

Fix: Use <picture> element with WebP/AVIF sources and PNG fallback.

<picture>
  <source type="image/avif" srcset="/public/hero.avif">
  <source type="image/webp" srcset="/public/hero.webp">
  <img src="/public/hero.png" alt="Hero banner" width="800" height="400" fetchpriority="high">
</picture>

WP-10: @font-face missing font-display

File: index.html, lines 13-17

Fix: Add font-display: swap (or optional for minimal CLS).

@font-face {
  font-family: "CustomFont";
  src: url("/fonts/custom.woff2") format("woff2");
  font-display: swap;
}

WP-11: Critical font not preloaded

File: index.html, <head> section

Fix: Add a preload link for the font file.

<link rel="preload" href="/fonts/custom.woff2" as="font" type="font/woff2" crossorigin>
