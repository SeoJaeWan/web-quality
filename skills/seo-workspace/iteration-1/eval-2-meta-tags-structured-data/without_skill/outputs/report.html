1. Meta Tag Issues (8 findings)

#SeverityLocationIssueRecommendation
1 HIGH <head> Missing <meta name="description"> tag. This is one of the most critical on-page SEO elements; search engines use it for snippet generation. Add a meta description of 150-160 characters that accurately summarizes the page and includes target keywords.
2 MEDIUM <head> Missing Open Graph meta tags (og:title, og:description, og:image, og:url, og:type). Shared links on social platforms will not display rich previews. Add OG tags. At minimum: og:title, og:description, og:image, and og:url.
3 MEDIUM <head> Missing Twitter Card meta tags. Twitter/X shares will lack rich card rendering. Add <meta name="twitter:card" content="summary_large_image"> and corresponding title, description, image tags.
4 LOW <head> Missing <link rel="canonical"> tag. Without it, search engines may index duplicate URLs (e.g., with query parameters or trailing slashes). Add a self-referencing canonical URL: <link rel="canonical" href="https://example.com/">.
5 LOW <head> Missing <meta name="robots"> tag. While the default is index/follow, explicit declaration is a best practice. Add <meta name="robots" content="index, follow">.
6 LOW <title> (line 6) Title is generic: "Sample Page". Non-descriptive titles reduce click-through rates and miss keyword targeting opportunities. Use a descriptive, keyword-rich title of 50-60 characters. Format: "Primary Keyword - Brand Name".
7 LOW <html> (line 2) No hreflang tags present. If the site targets multiple regions or languages, this is needed for proper localization signals. Add <link rel="alternate" hreflang="en" href="..."> if multilingual content exists.
8 LOW <head> Missing favicon declaration. Favicons appear in browser tabs, bookmarks, and Google search results. Add <link rel="icon" href="/favicon.ico"> and <link rel="apple-touch-icon" href="/apple-touch-icon.png">.

2. Structured Data Issues (4 findings)

#SeverityLocationIssueRecommendation
9 HIGH <head> No JSON-LD structured data is present anywhere in the document. Search engines cannot generate rich results (rich snippets, knowledge panels) without structured data. Add a <script type="application/ld+json"> block with at minimum a WebPage schema including name, description, and url.
10 HIGH Page-level Missing Organization or WebSite schema markup. Without this, Google cannot associate the page with a brand entity or enable sitelinks search box. Add Organization schema with name, url, logo, and sameAs (social profiles). Add WebSite schema with SearchAction for sitelinks search box eligibility.
11 MEDIUM Page-level No BreadcrumbList structured data. Breadcrumbs in search results improve CTR and help users understand site hierarchy. Add BreadcrumbList JSON-LD with itemListElement entries reflecting the page's position in the site hierarchy.
12 LOW Page-level No SearchAction or SiteNavigationElement schema. These enable enhanced SERP features like sitelinks search box. Consider adding SearchAction within a WebSite schema to enable the sitelinks searchbox.

3. SEO-Adjacent Issues (3 findings)

13 MEDIUM <img> (line 30) Image /public/logo.png is missing the alt attribute. Missing alt text harms image SEO and accessibility (a Core Web Vitals / page experience signal). Add a descriptive alt attribute, e.g., alt="Company logo".

Recommended JSON-LD Structured Data (Example)

Below is a minimal JSON-LD block that addresses the structured data gaps identified above:

<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@graph": [
    {
      "@type": "Organization",
      "name": "Example Company",
      "url": "https://example.com",
      "logo": "https://example.com/public/logo.png",
      "sameAs": [
        "https://twitter.com/example",
        "https://www.facebook.com/example"
      ]
    },
    {
      "@type": "WebSite",
      "name": "Example Company",
      "url": "https://example.com",
      "potentialAction": {
        "@type": "SearchAction",
        "target": "https://example.com/search?q={search_term_string}",
        "query-input": "required name=search_term_string"
      }
    },
    {
      "@type": "WebPage",
      "name": "Welcome to Example Company",
      "description": "Discover our latest deals and learn about our company.",
      "url": "https://example.com/",
      "breadcrumb": {
        "@type": "BreadcrumbList",
        "itemListElement": [
          {
            "@type": "ListItem",
            "position": 1,
            "name": "Home",
            "item": "https://example.com/"
          }
        ]
      }
    }
  ]
}
</script>
