SEO-02: title 태그 (Line 6)

현재 11자로 50-60자 권장에 크게 미달합니다. 페이지 핵심 키워드를 포함한 고유한 제목으로 변경하세요.

<!-- Before -->
<title>Sample Page</title>

<!-- After -->
<title>Premium Sample Products - Free Shipping & 30-Day Returns | BrandName</title>

SEO-05: meta description 누락

150-160자의 매력적인 메타 설명을 추가하세요. 주요 키워드와 CTA를 포함합니다.

<meta name="description" content="Premium quality sample products with free shipping.
Browse our collection of 500+ items. Rated 4.9/5 by 10,000+ customers. Order today and save 20%.">

SEO-06: canonical URL 누락

<head> 내에 canonical URL을 추가하여 중복 콘텐츠 문제를 방지하세요.

<link rel="canonical" href="https://example.com/">

SEO-07: JSON-LD 구조화 데이터 누락

페이지 유형에 맞는 JSON-LD 구조화 데이터를 추가하세요.

<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "WebPage",
  "name": "Sample Page",
  "url": "https://example.com/"
}
</script>

WP-01: render-blocking 인라인 스크립트 (Lines 37-44)

인라인 스크립트를 외부 파일로 분리하고 defer를 적용하거나, DOMContentLoaded 이벤트를 사용하세요.

<!-- Option 1: External script with defer -->
<script defer src="/js/app.js"></script>

<!-- Option 2: Wrap in DOMContentLoaded -->
<script>
document.addEventListener('DOMContentLoaded', function() {
  var label = document.getElementById("label");
  var userInput = location.hash.slice(1);
  if (label) {
    label.textContent = "Hello, " + userInput;  // textContent for XSS safety
  }
});
</script>

WP-02: LCP 이미지 fetchpriority + preload 누락 (Line 27)

히어로 이미지에 fetchpriority="high"를 추가하고, <head>에 preload 링크를 추가하세요.

<!-- head 내에 추가 -->
<link rel="preload" href="/public/hero.png" as="image" fetchpriority="high">

<!-- img 태그 수정 -->
<img src="/public/hero.png" alt="Hero banner" width="800" height="400" fetchpriority="high">

WP-03: render-blocking CSS (Line 10)

Critical CSS를 인라인 처리하고, 외부 CSS는 비동기로 로드하세요.

<!-- Critical CSS inline -->
<style>/* Above-fold critical styles here */</style>

<!-- Non-critical CSS deferred -->
<link rel="preload" href="https://cdn.example.com/styles.css" as="style"
      onload="this.onload=null;this.rel='stylesheet'">
<noscript><link rel="stylesheet" href="https://cdn.example.com/styles.css"></noscript>

WP-04: promo.png width/height 누락 (Line 33)

이미지에 width와 height 속성을 추가하여 CLS를 방지하세요.

<img src="/public/promo.png" alt="Promotional banner" width="800" height="400">

WP-05: loading="lazy" 누락 (Lines 30, 33)

비-LCP 이미지에 loading="lazy"를 추가하여 초기 로드를 최적화하세요.

<img src="/public/logo.png" width="120" height="40" loading="lazy" alt="Logo">
<img src="/public/promo.png" alt="Promotional banner" width="800" height="400" loading="lazy">

WP-06: 현대 이미지 포맷 미사용

WebP 또는 AVIF 포맷을 <picture> 요소와 함께 사용하여 파일 크기를 줄이세요.

<picture>
  <source type="image/avif" srcset="/public/hero.avif">
  <source type="image/webp" srcset="/public/hero.webp">
  <img src="/public/hero.png" alt="Hero banner" width="800" height="400" fetchpriority="high">
</picture>

WP-10: font-display 누락 (Lines 13-17)

@font-face에 font-display: swap을 추가하여 FOIT를 방지하세요.

@font-face {
  font-family: "CustomFont";
  src: url("/fonts/custom.woff2") format("woff2");
  font-display: swap;  /* or 'optional' for zero CLS */
}

WP-11: 중요 폰트 preload 누락

<head> 상단에 폰트 preload를 추가하여 다운로드를 앞당기세요.

<link rel="preload" href="/fonts/custom.woff2" as="font" type="font/woff2" crossorigin>
