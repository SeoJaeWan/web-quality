수정 가이드

❌ WP-04: 이미지 width/height 누락 (CLS 방지)

파일: dashboard.tsx line 50

문제: <img> 태그에 크기 정보가 없어 이미지 로드 시 레이아웃이 밀림

수정 방법:

<!-- Before -->
<img src="/public/promo.png" alt="Promo" className="mt-8" />

<!-- After -->
<img src="/public/promo.png" alt="Promo" className="mt-8"
     width="800" height="400"
     style={{ aspectRatio: '2/1' }} />

❌ WP-06: 현대 이미지 포맷 미사용

❌ WP-09: 레이아웃 스래싱

파일: dashboard.tsx lines 11-13

문제: offsetHeight 읽기 → style.minHeight 쓰기 연속 호출로 강제 리플로우 2회 발생

수정 방법: 읽기와 쓰기를 분리하거나 requestAnimationFrame 활용

useEffect(() => {
  if (containerRef.current) {
    const h = containerRef.current.offsetHeight; // 읽기
    requestAnimationFrame(() => {
      if (containerRef.current) {
        containerRef.current.style.minHeight = h + "px"; // 쓰기 (다음 프레임)
      }
    });
  }
}, []);
