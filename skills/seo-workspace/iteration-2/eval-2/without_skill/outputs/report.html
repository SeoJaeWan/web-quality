Recommendation: Use a descriptive title (50-60 characters) containing primary keywords, e.g. <title>Product Name - Primary Keyword | Brand Name</title>.
Critical SEO-002 Missing Meta Description

No <meta name="description"> tag found. The meta description is a key factor in click-through rate from search results. Without it, search engines will auto-generate a snippet which may not be optimal.

Recommendation: Add <meta name="description" content="..."> with a compelling 150-160 character summary that includes target keywords and a call to action.
Critical SEO-003 No Structured Data (JSON-LD)

No JSON-LD or Schema.org structured data found on the page. Structured data enables rich snippets in search results (ratings, breadcrumbs, FAQs, etc.) and helps search engines understand page content.

Recommendation: Add JSON-LD structured data in a <script type="application/ld+json"> block. At minimum, include WebPage or WebSite schema. Example: 
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "WebPage",
  "name": "Page Title",
  "description": "Page description",
  "url": "https://example.com/"
}
</script>

Major Issues

Major SEO-004 Missing Robots Meta Tag

No <meta name="robots"> tag is present. While search engines default to "index, follow", explicitly declaring the directive is a best practice for controlling crawl behavior.

Recommendation: Add <meta name="robots" content="index, follow"> to explicitly declare indexing intent.
Major SEO-005 Missing Canonical URL

No <link rel="canonical"> tag found. Without a canonical URL, duplicate content issues can arise from URL variations (www vs non-www, trailing slashes, query parameters).

Recommendation: Add <link rel="canonical" href="https://example.com/"> pointing to the preferred version of the page URL.
Major SEO-006 Missing Open Graph Meta Tags

No Open Graph tags found (og:title, og:description, og:image, og:url, og:type). These tags control how the page appears when shared on social media platforms like Facebook, LinkedIn, and others.

Recommendation: Add Open Graph meta tags: 
<meta property="og:type" content="website">
<meta property="og:title" content="Page Title">
<meta property="og:description" content="Page description">
<meta property="og:url" content="https://example.com/">
<meta property="og:image" content="https://example.com/image.png">
Major SEO-007 Missing Twitter Card Meta Tags

No Twitter Card tags found (twitter:card, twitter:title, twitter:description). These control the appearance of links shared on Twitter/X.

Recommendation: Add Twitter Card meta tags: 
<meta name="twitter:card" content="summary_large_image">
<meta name="twitter:title" content="Page Title">
<meta name="twitter:description" content="Page description">
<meta name="twitter:image" content="https://example.com/image.png">

No BreadcrumbList schema found. Breadcrumb structured data improves SERP appearance by showing the page's position within the site hierarchy.
