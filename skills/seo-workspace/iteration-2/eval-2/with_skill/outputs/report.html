수정 가이드

SEO-02: title 태그 개선 (Critical)

현재 <title>Sample Page</title> (11자)는 너무 짧고 비서술적입니다. 50-60자 이내로 주요 키워드를 포함한 고유한 타이틀을 작성하세요.

<title>프리미엄 위젯 쇼핑몰 | 무료배송 & 최저가 보장 | Example Store</title>

SEO-05: meta description 추가 (High)

검색 결과 스니펫에 표시될 설명이 없습니다. 150-160자 이내로 페이지 내용을 요약하고 행동 유도 문구를 포함하세요.

<meta name="description" content="프리미엄 위젯을 최저가로 만나보세요. 무료배송, 30일 반품 보장. 10,000명 이상의 고객이 선택한 신뢰할 수 있는 쇼핑몰입니다. 지금 주문하면 20% 할인!">

SEO-06: canonical URL 추가 (High)

중복 콘텐츠 문제를 방지하기 위해 self-referencing canonical URL을 설정하세요.

<link rel="canonical" href="https://example.com/">

SEO-07: JSON-LD 구조화 데이터 추가 (High)

검색 결과에서 리치 스니펫(별점, 가격 등) 노출을 위해 JSON-LD 구조화 데이터를 추가하세요. 홈페이지의 경우 Organization 스키마가 적합합니다.

<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Organization",
  "name": "Example Company",
  "url": "https://example.com",
  "logo": "https://example.com/logo.png",
  "contactPoint": {
    "@type": "ContactPoint",
    "telephone": "+1-555-123-4567",
    "contactType": "customer service"
  }
}
</script>

WP-01: 인라인 스크립트 최적화

line 37의 인라인 스크립트를 외부 파일로 분리하고 defer 속성을 추가하세요. 또한, innerHTML 사용은 XSS 위험이 있으므로 textContent로 대체하세요.

<script defer src="/js/main.js"></script>

WP-02: LCP 이미지 우선순위 설정

hero.png에 fetchpriority="high"를 추가하고, head에 preload를 선언하세요.

<!-- head 내부 -->
<link rel="preload" href="/public/hero.png" as="image" fetchpriority="high">

<!-- body 내부 -->
<img src="/public/hero.png" alt="Hero banner" width="800" height="400" fetchpriority="high">

WP-03: Critical CSS 처리

line 10의 외부 CSS를 비동기로 로딩하고, above-fold에 필요한 CSS는 인라인으로 삽입하세요.

<style>/* above-fold에 필요한 핵심 CSS */</style>
<link rel="preload" href="https://cdn.example.com/styles.css" as="style" onload="this.onload=null;this.rel='stylesheet'">
<noscript><link rel="stylesheet" href="https://cdn.example.com/styles.css"></noscript>

WP-04: promo.png에 width/height 추가

line 33: CLS 방지를 위해 이미지에 width/height 또는 aspect-ratio를 지정하세요.

<img src="/public/promo.png" alt="Promotional banner" width="800" height="400">

WP-06: 현대 이미지 포맷 도입

PNG 대신 WebP/AVIF 포맷을 <picture> 요소로 제공하세요.

<picture>
  <source type="image/avif" srcset="/public/hero.avif">
  <source type="image/webp" srcset="/public/hero.webp">
  <img src="/public/hero.png" alt="Hero banner" width="800" height="400" fetchpriority="high">
</picture>

WP-10: font-display 추가

line 13-17: @font-face에 font-display: swap을 추가하여 폰트 로딩 중 텍스트가 보이도록 하세요.

@font-face {
  font-family: "CustomFont";
  src: url("/fonts/custom.woff2") format("woff2");
  font-display: swap;
}

WP-11: 폰트 preload 추가

head 영역에 중요 폰트를 preload하세요.

<link rel="preload" href="/fonts/custom.woff2" as="font" type="font/woff2" crossorigin>
