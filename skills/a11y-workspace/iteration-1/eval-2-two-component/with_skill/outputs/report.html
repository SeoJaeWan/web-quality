불합격 (❌)

항목 10. 키보드 사용 보장 — StatsCard.tsx:17

문제: <div onClick={handleClick}> — 비대화형 요소에 onClick만 사용하여 키보드 사용자가 해당 기능에 접근할 수 없음. tabIndex, role, onKeyDown이 모두 누락됨

수정: <div>를 <button type="button" onClick={handleClick} className="rounded-xl border ...">로 교체. 또는 <div role="button" tabIndex={0} onClick={handleClick} onKeyDown={(e) => { if (e.key === 'Enter' || e.key === ' ') handleClick(); }}> 추가

항목 17. 반복 영역 건너뛰기 — Navbar.tsx / layout.tsx

문제: 페이지 최상단에 건너뛰기 링크(skip navigation)가 없어 키보드 사용자가 반복되는 내비게이션 영역을 매번 탐색해야 함

수정: layout.tsx의 <body> 직후에 <a href="#main-content" style="position:absolute;top:-40px;left:0;z-index:100" onFocus="this.style.top='0'">본문 바로가기</a> 추가, <main>에 id="main-content" 부여

항목 32. 마크업 오류 방지 — StatsCard.tsx:13

문제: id="stat-card"가 하드코딩되어 있어 컴포넌트가 페이지에 여러 번 렌더링되면 동일 ID가 중복됨 (HTML 명세 위반)

수정: id={`stat-card-${testId}`}로 고유 ID 생성하거나, id 속성 자체를 제거하고 data-testid만 사용

시맨틱 HTML: 대화형 요소 — StatsCard.tsx:10-18

문제: 클릭 가능한 카드가 <div onClick>로 구현됨. 의미적으로 버튼 역할이나 <button> 요소를 사용하지 않음

수정: <button type="button" onClick={handleClick} className="...">로 교체하여 키보드 접근성과 시맨틱 마크업을 동시에 확보
