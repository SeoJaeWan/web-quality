Accessibility

index.html
SeverityLineIssueRecommendation
Critical 30 Image missing alt attribute (logo.png). Screen readers cannot describe this image to users. Add a descriptive alt attribute, or alt="" with role="presentation" if purely decorative.
Major 20–45 No landmark regions (<header>, <main>, <nav>, <footer>). Assistive technology users cannot quickly navigate page structure. Wrap content in semantic landmark elements: <header>, <main>, etc.
Major 6 lang="en" is set, but the page has no skip-navigation link. Keyboard-only users must tab through all links. Add a “Skip to content” link before the first focusable element pointing to <main id="content">.
Card.tsx
SeverityLineIssueRecommendation
Critical 25–27 Duplicate id="card-item" when Card is rendered in a list. Violates HTML spec (IDs must be unique) and breaks aria-* references and document.getElementById. Remove the static id or generate a unique id per instance (e.g., via a useId() hook or prop).
Critical 29 Click handler on a <div> without keyboard support. The element is not focusable and cannot be activated via Enter/Space, failing WCAG 2.1.1 (Keyboard). Either use a <button> element, or add role="button", tabIndex={0}, and an onKeyDown handler for Enter/Space.
Minor 31 Heading level <h3> used inside Card. If Card is placed outside of an <h2> context, heading order may be broken (WCAG 1.3.1). Consider making the heading level configurable via an as prop, or document the expected heading hierarchy.

SEO

index.html
SeverityLineIssueRecommendation
Major 7 Missing <meta name="description">. Search engines use this for snippet text; its absence may hurt click-through rate. Add <meta name="description" content="..."> with a 120–160 character summary of the page.
Major 8 No structured data (JSON-LD). The page provides no machine-readable schema for rich-result eligibility. Add a <script type="application/ld+json"> block with at minimum WebPage or Organization schema.
Minor N/A No canonical URL (<link rel="canonical">) specified. Duplicate-content signals may be diluted. Add <link rel="canonical" href="https://..."> in <head>.

Performance

index.html
SeverityLineIssueRecommendation
Major 10 Render-blocking CSS loaded synchronously in <head>. This delays First Contentful Paint. Use media="print" onload="this.media='all'" pattern, or inline critical CSS and load the rest asynchronously.
Major 13–17 @font-face missing font-display: swap. Text is invisible until the custom font loads (FOIT), hurting LCP. Add font-display: swap; to the @font-face rule.
Info 33 Image promo.png lacks explicit width/height attributes, contributing to Cumulative Layout Shift (CLS). Add width and height attributes matching the image's intrinsic dimensions.
