수정 가이드

A-17: 반복 영역 건너뛰기

건너뛰기 링크(skip navigation)를 <main> 요소 전에 추가하여 키보드 사용자가 반복되는 네비게이션 영역을 건너뛸 수 있도록 합니다.

<a href="#main-content" className="sr-only focus:not-sr-only">
  본문 바로가기
</a>
<main id="main-content" className="p-4">
  ...
</main>
A-23: 레이블과 네임 — 아이콘 버튼 접근성

SVG만 포함된 아이콘 버튼에 aria-label을 추가하여 스크린 리더 사용자가 버튼의 용도를 파악할 수 있도록 합니다.

<button aria-label="알림">
  <svg width="16" height="16" viewBox="0 0 16 16">
    <circle cx="8" cy="8" r="8" />
  </svg>
</button>
<button aria-label="설정">
  <svg width="16" height="16" viewBox="0 0 16 16">
    <rect width="16" height="16" />
  </svg>
</button>
A-32: 마크업 오류 방지 — key prop 누락

React 리스트 렌더링 시 각 요소에 고유한 key prop을 제공해야 합니다. 인덱스 대신 고유 식별자를 사용하는 것을 권장합니다.

{items.map((item, index) => (
  <div key={item} className="border p-2">{item}</div>
))}
WP-04: img width/height 또는 aspect-ratio (CLS 방지)

이미지에 width와 height 속성을 추가하여 브라우저가 이미지 로드 전에 공간을 예약하도록 합니다.

<img src="/public/promo.png" alt="프로모션 배너 - 신규 기능 안내"
     width="800" height="400"
     loading="lazy"
     className="mt-8" />
WP-05: loading="lazy" (below-fold 이미지)

화면 아래쪽(below-fold)에 위치한 이미지에 loading="lazy"를 추가하여 뷰포트에 가까워질 때만 로드하도록 합니다.

<img src="/public/promo.webp" alt="프로모션 배너"
     loading="lazy" decoding="async"
     width="800" height="400"
     className="mt-8" />
WP-06: 현대 이미지 포맷 (WebP/AVIF)

.png 대신 WebP/AVIF 포맷을 사용하면 30-80% 파일 크기를 절감할 수 있습니다. <picture> 요소로 포맷 폴백을 제공합니다.

<picture>
  <source srcSet="/public/promo.avif" type="image/avif" />
  <source srcSet="/public/promo.webp" type="image/webp" />
  <img src="/public/promo.png" alt="프로모션 배너"
       width="800" height="400"
       loading="lazy" decoding="async" />
</picture>
WP-09: 레이아웃 스래싱 제거

offsetHeight 읽기와 style.minHeight 쓰기를 분리하여 강제 리플로우를 방지합니다. requestAnimationFrame을 사용하여 쓰기를 다음 프레임으로 지연시킵니다.

useEffect(() => {
  if (containerRef.current) {
    const h = containerRef.current.offsetHeight;
    requestAnimationFrame(() => {
      if (containerRef.current) {
        containerRef.current.style.minHeight = h + "px";
      }
    });
  }
}, []);
WP-18: 동적 콘텐츠 공간 예약 (CLS 방지)

500ms 후 동적으로 삽입되는 아이템 목록에 대해 미리 공간을 예약하여 레이아웃 시프트를 방지합니다.

<section ref={containerRef} className="grid gap-4" style={{ minHeight: '200px' }}>
  <h2 className="text-lg font-semibold">Overview</h2>
  {items.length === 0 ? (
    <div className="animate-pulse bg-gray-200 h-32 rounded" />
  ) : (
    items.map((item) => (
      <div key={item} className="border p-2">{item}</div>
    ))
  )}
</section>
