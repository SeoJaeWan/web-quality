30 Critical Accessibility <img> missing alt attribute (logo.png). Screen readers cannot describe the image, violating WCAG 1.1.1 (Non-text Content). Add a descriptive alt attribute, e.g. alt="Company logo". If decorative, use alt="" with role="presentation".
38-43 Critical Security innerHTML is assigned with unsanitized user input from location.hash. This is a DOM-based XSS vulnerability. Use textContent instead of innerHTML, or sanitize input with a library like DOMPurify before inserting into the DOM.
7 Major SEO Missing <meta name="description">. Search engines rely on this for snippet generation. Add <meta name="description" content="..."> with a concise page summary (120-160 characters).
8 Minor SEO No JSON-LD structured data present. This limits rich-result eligibility in search engines. Add a <script type="application/ld+json"> block with appropriate schema (e.g. WebPage, Organization).
10 Major Performance Render-blocking external stylesheet loaded synchronously in <head>. This delays First Contentful Paint (FCP). Inline critical CSS and load the remaining stylesheet asynchronously using <link rel="preload" as="style" onload="this.rel='stylesheet'"> or move it after the fold.
13-17 Major Performance @font-face declaration missing font-display: swap. Custom fonts block text rendering until loaded (FOIT). Add font-display: swap; to the @font-face rule to show fallback text while the custom font loads.
33 Minor Performance <img> for promo.png is missing explicit width and height attributes. This causes Cumulative Layout Shift (CLS). Add width and height attributes that match the image's intrinsic dimensions.
37-44 Minor Performance Inline <script> in <body> without defer or async blocks HTML parsing. Move the script to an external file and use defer, or place it at the end of <body> (already at end, but externalizing is preferred).
20 Info Accessibility Page has no landmark elements (<header>, <nav>, <main>, <footer>). Landmark regions improve screen reader navigation. Wrap content in semantic landmark elements (e.g. <main>, <header>).
6 Info SEO Page <title> is generic ("Sample Page"). A descriptive title improves click-through rate in search results. Use a specific, keyword-rich title (50-60 characters) that describes the page content.

LineSeverityCategoryIssueRecommendation
27 Critical Accessibility Hardcoded id="card-item" on a reusable component. When rendered in a list, multiple elements share the same id, violating the HTML spec and breaking accessibility (ARIA references, document.getElementById). Generate unique IDs using React.useId() or accept an id prop. Alternatively, remove the id if not needed.
29 Major Accessibility <div onClick> is not keyboard-accessible. The click handler cannot be triggered via keyboard (Enter/Space), violating WCAG 2.1.1 (Keyboard). Either use a <button> element instead, or add role="button", tabIndex={0}, and an onKeyDown handler for Enter/Space keys.
17-20 Minor Maintainability document.querySelector used inside a React component to manipulate the DOM directly. This bypasses React's virtual DOM and can cause unexpected behavior, especially with multiple Card instances. Use a useRef hook to reference the target element, and manage styles via React state or CSS classes.
13 Minor Maintainability console.log left in production code. This clutters the browser console and may leak information. Remove the console.log or replace with a conditional debug logger that is stripped in production builds.
