WP-09: 레이아웃 스래싱 — offsetHeight 읽기/쓰기 혼합

dashboard.tsx:12-13 — offsetHeight 읽기 직후 style.minHeight 쓰기가 연속으로 발생하여 브라우저가 두 번 리플로우합니다. requestAnimationFrame으로 쓰기를 분리하세요.

useEffect(() => {
  if (containerRef.current) {
    const h = containerRef.current.offsetHeight;
    requestAnimationFrame(() => {
      if (containerRef.current) {
        containerRef.current.style.minHeight = h + "px";
      }
    });
  }
}, []);
