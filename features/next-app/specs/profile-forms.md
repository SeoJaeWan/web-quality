# 프로필 및 복잡한 폼 테스트 플랜

**Base URL:** http://localhost:3000
**전제조건:** 로그인 완료 상태

---

### 1. 프로필 조회 및 수정

**Seed:** `seed.spec.ts`

#### 1.1 프로필 조회
**Steps:**
1. /profile 페이지로 이동
2. 프로필 정보(이름, 이메일) 표시 확인

**Expected:** 가입 시 입력한 이름과 이메일이 표시됨

#### 1.2 프로필 수정
**Steps:**
1. /profile 페이지에서 "프로필 수정" 버튼 클릭
2. 이름을 "수정된이름"으로 변경
3. 자기소개에 "안녕하세요" 입력
4. "저장" 버튼 클릭
5. 저장 성공 메시지 확인

**Expected:** 프로필 정보가 업데이트되고 "프로필이 저장되었습니다" 메시지 표시

---

### 2. 파일 업로드

**Seed:** `seed.spec.ts`

#### 2.1 유효한 이미지 업로드
**Steps:**
1. /profile 수정 모드 진입
2. 아바타 "파일 선택" 버튼 클릭
3. 유효한 이미지 파일(JPG, PNG) 선택
4. 선택된 파일명 표시 확인

**Expected:** "선택됨: filename.jpg" 형태로 파일명 표시

#### 2.2 허용되지 않는 파일 형식
**Steps:**
1. /profile 수정 모드 진입
2. 아바타에 .txt 파일 선택
3. 에러 메시지 확인

**Expected:** "허용되지 않는 파일 형식입니다" 에러 표시

---

### 3. 날짜 선택기

**Seed:** `seed.spec.ts`

#### 3.1 생년월일 입력
**Steps:**
1. /profile 수정 모드 진입
2. 생년월일 필드에 날짜 입력
3. 저장 후 확인

**Expected:** 입력한 날짜가 프로필에 표시됨

---

### 4. 멀티셀렉트 드롭다운

**Seed:** `seed.spec.ts`

#### 4.1 기술 스택 선택
**Steps:**
1. /profile 수정 모드 진입
2. 기술 스택 드롭다운 클릭
3. "JavaScript", "React" 체크
4. 선택된 항목이 칩으로 표시되는지 확인

**Expected:** "JavaScript", "React" 칩이 표시되고 "2개 선택됨" 표시

#### 4.2 선택 해제
**Steps:**
1. 기술 스택에서 칩의 X 버튼 클릭
2. 해당 항목이 제거되는지 확인

**Expected:** 해당 기술 스택이 칩에서 제거됨

#### 4.3 전체 선택 후 드롭다운 닫기
**Steps:**
1. 드롭다운 외부 영역 클릭
2. 드롭다운이 닫히는지 확인

**Expected:** 드롭다운이 닫히고 선택한 항목은 유지됨
